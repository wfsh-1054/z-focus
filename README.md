# Z-Focus | AI Drawing Workstation

Z-Focus is a high-performance AI creative workstation. It bridges the gap between complex prompt engineering and intuitive creativity through advanced reasoning and high-fidelity visual synthesis. Powered by Pollinations.

## üåü Core Features

- **Advanced Reasoning**: Leverages state-of-the-art LLMs to transform simple keywords into professional-grade, descriptive 8K prompts.
- **Instructional Editing**: True natural language image modification. Instead of re-typing prompts, simply tell the AI to "add sunglasses" or "change the background to a forest."
- **Z-Image Synthesis Engine**: Access to Pollinations' dedicated high-fidelity image model, optimized for speed and aesthetic precision.
- **Professional Control Suite**:
  - **Aspect Ratios**: Support for Square, Landscape, Portrait, Wide, and Tall formats.
  - **Aesthetic Presets**: 10+ built-in styles including Cinematic, Anime, Neon Punk, and 3D Model.
  - **Canvas Archive**: Local session history for tracking and revisiting previous iterations.
